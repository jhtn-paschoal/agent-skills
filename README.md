# Agent Skills - Governança Central

Repositório central de diretrizes arquiteturais para o orquestrador. Este repositório atua como a única fonte de verdade (Single Source of Truth) para os padrões de engenharia de software aplicados pela inteligência artificial.

## Instalação de Skills

Para injetar uma skill de arquitetura em um novo repositório, execute o comando abaixo no terminal da raiz do projeto alvo:

```bash
npx skills add [https://github.com/jhtn-paschoal/agent-skills](https://github.com/jhtn-paschoal/agent-skills) --skill <nome-da-skill>
